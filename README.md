# Rule-Based Coding Troubleshooting Coach

## Overview

The Rule-Based Coding Troubleshooting Coach is a simple Python program that helps users troubleshoot common programming problems. The system asks the user questions, follows predefined rules, and gives helpful debugging advice based on the user’s answers.

This project demonstrates how early AI systems worked before machine learning. Instead of learning from data, the program uses human-created rules, decision trees, and `if/elif/else` logic to simulate intelligent behavior.

## Project Purpose

The purpose of this project is to understand how rule-based artificial intelligence systems work. A rule-based system makes decisions by following predefined logic such as:


IF the user selects SyntaxError
THEN suggest checking for missing colons, parentheses, quotation marks, or indentation.
